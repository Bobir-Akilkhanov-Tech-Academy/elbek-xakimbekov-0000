1) git checkout -b branch_name (создаст ветку и перейдёт на неё)

2) git branch branch_name (создаст ветку)

3) git checkout branch_name (перейдёт на ветку)

4) git branch (выведет список всех веток)

5) git branch -d branch_name

6) git push -u origin branch_name (опубликовать ветку удаленно)

7) git push -d origin branch_name (удалить ветку на удаленном репозитории)
